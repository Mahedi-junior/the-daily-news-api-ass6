#What is the difference between arrow function and regular function?

Ans: Inside the body of a regular function, arguments is a special array-like object containing the list of arguments with which the function has been invoked. On the other side, no arguments special keyword is defined inside an arrow function. Again (same as with this value), the arguments object is resolved lexically: the arrow function accesses arguments from the outer function.
